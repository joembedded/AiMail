# Jo's AiMail

Ein einfaches Werkzeug zur Formatierung und Korrektur geschäftlicher E-Mails in minimalistischer Form.

## Funktionen

- Formatiert geschäftliche E-Mails in einem klaren, minimalistischen Stil
- Korrigiert Rechtschreibung und Grammatik
- Identifiziert potenzielle Probleme in der E-Mail
- Listet vorgenommene Änderungen auf

## Verwendung

1. Öffne `index.html` im Browser
2. Füge deine E-Mail in das Textfeld ein
3. Klicke auf "Korrigieren"
4. Erhalte die korrigierte Version mit detaillierten Änderungen

## Technische Details

- Frontend: HTML, CSS, JavaScript
- Backend: PHP mit OpenAI API-Integration
- Keine Installation erforderlich - einfach auf einem Webserver bereitstellen

## Konfiguration

Die API-Schlüssel werden in `secret/keys.inc.php` gespeichert. Eine Vorlage findest du in `secret/_dummy_keys.inc.php`.
